# 致爸爸妈妈的一封信

## 📝 简介

这是在十八岁成人礼上，写给爸爸妈妈的一封感谢信。

## 🎨 设计特点

- **温馨配色**：使用温暖的米色、粉色、棕色色调，营造家的温暖感
- **平滑动画**：信件内容按顺序淡入，配合延迟效果，营造阅读的节奏感
- **优雅排版**：采用信纸风格的布局，字体清晰易读
- **背景音乐**：自动播放温馨的背景音乐（可手动控制）
- **响应式设计**：支持手机、平板、电脑等多种设备
- **打印友好**：优化了打印样式，可以打印成纸质信件

## 📁 文件结构

```
my-website/
├── index.html              # 主页面（信件内容）
├── css/
│   └── style.css          # 样式文件
├── js/
│   └── main.js            # JavaScript（动画和音频控制）
├── audio.wav              # 背景音乐（可以更换成更温馨的音乐）
├── index.html.backup      # 原始版本备份
├── index.html.letter-backup  # 信件版本备份
└── readme.md              # 说明文件
```

## 🚀 使用方法

### 在浏览器中打开

1. 直接用浏览器打开 `index.html` 文件
2. 页面会自动播放背景音乐（3秒后开始）
3. 信件内容会按顺序淡入显示
4. 点击右下角的音乐按钮可以控制音乐播放/暂停
5. 按空格键可以暂停/继续动画

### 在线部署

可以将整个文件夹上传到任何静态网站托管服务，如：
- GitHub Pages
- Vercel
- Netlify
- 自己的服务器

### 打印成纸质信件

1. 在浏览器中打开网页
2. 按下 Ctrl+P (Windows) 或 Cmd+P (Mac)
3. 选择打印
4. 会自动优化打印样式，移除动画和音乐按钮

## 🎵 更换背景音乐

如果想要更换背景音乐：

1. 找一首温馨、感人的音乐（建议时长3-5分钟）
2. 将音乐文件重命名为 `audio.wav`（或修改 HTML 中的文件名）
3. 替换 `my-website/` 目录下的 `audio.wav` 文件

推荐的音乐类型：
- 温馨的钢琴曲
- 感人的轻音乐
- 家庭主题的背景音乐

## ✏️ 修改信件内容

如果需要修改信件内容：

1. 打开 `index.html` 文件
2. 找到 `<main class="letter-content">` 部分
3. 修改 `<p class="letter-text">` 标签中的文字内容
4. 保存文件

## 🎨 自定义样式

如果需要修改颜色、字体等样式：

1. 打开 `css/style.css` 文件
2. 找到对应的部分进行修改
3. 保存文件

主要可修改的部分：
- 颜色方案（搜索 `#` 开头的颜色代码）
- 字体大小（搜索 `font-size`）
- 间距（搜索 `margin`, `padding`）
- 动画效果（搜索 `@keyframes`）

## 📱 响应式设计

网页已经优化了移动端显示：
- 在手机上会自动调整字体大小和间距
- 内容会自适应屏幕宽度
- 触摸操作友好

## 💡 小贴士

- 首次打开网页时，浏览器可能会询问是否允许自动播放音频，请选择允许
- 如果音乐无法自动播放，可以点击右下角的音乐按钮手动播放
- 可以按空格键暂停动画，方便分段阅读
- 建议使用 Chrome、Firefox、Safari 等现代浏览器以获得最佳体验
